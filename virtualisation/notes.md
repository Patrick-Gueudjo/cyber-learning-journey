# Notes du cours — Virtualisez vos environnements de travail

## Hyperviseurs
- Type 1 : Bare metal (ESXi, Hyper-V Server,KVM,Xen,Proxmox VE)
- Type 2 : Hosted (VirtualBox, VMware Workstation,VMware Fusion(macOS),QEMU,)

## Réseaux virtuels
- NAT
- Reseau Nat
- Bridge (accès direct au réseau physique)
- Host-only (réseau local isolé)
- Internal network

## Snapshots
- Sauvegarde de l’état d’une VM à un instant T
- Utile pour les tests ou les labs cybersécurité

## Gestion des VM
- CPU, RAM, Storage
- Allocation dynamique vs fixe
- ISO boot
